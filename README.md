# 💰 Expense Tracker

A modern Expense Tracker web application built using React and Vite. This application helps users manage their daily expenses with an interactive dashboard, charts, reports, and budget tracking.

---

## 🚀 Features

- 🔐 User Login & Signup
- 📊 Dashboard with Summary Cards
- ➕ Add New Expense
- ✏️ Edit Expense
- 🗑️ Delete Expense
- 🔍 Search Expenses
- 📂 Filter Expenses by Category
- 📈 Expense Overview Chart
- 📄 PDF Report Download
- 💰 Budget Alert
- 🌙 Dark Mode
- 👤 User Profile
- 💾 Local Storage Support

---

## 🛠️ Technologies Used

- React.js
- Vite
- JavaScript (ES6+)
- CSS3
- Recharts
- jsPDF
- React Icons
- React Toastify
- Local Storage

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/supreya-14/expense-tracker-react.git
```

Go to project folder

```bash
cd expense-tracker-react
```

Install dependencies

```bash
npm install
```

Run the project

```bash
npm run dev
```
